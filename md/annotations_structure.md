# annotations.json 结构说明

`annotations.json` 文件用于存储视频标注数据，其采用三层嵌套的 JSON 结构。

## 结构层级

1.  **第一层：视频层**
    *   **Key**: `视频文件名` (String)
        *   示例：`"Camera1-1.avi"`
    *   **Value**: 包含该视频下所有标注帧的对象（字典）。

2.  **第二层：帧层**
    *   **Key**: `帧号` (String)
        *   示例：`"300"`
        *   注意：帧号为**字符串**格式，且通常为 **1-based**（从 1 开始计数）。
    *   **Value**: 该帧内所有目标检测框的列表（数组）。

3.  **第三层：目标框层**
    *   **Value**: `[x_min, y_min, x_max, y_max]` (List of Integer)
    *   **含义**: 一个包含 4 个整数的数组，代表矩形框的像素坐标。
        *   `x_min`: 左上角 X 坐标
        *   `y_min`: 左上角 Y 坐标
        *   `x_max`: 右下角 X 坐标
        *   `y_max`: 右下角 Y 坐标

## JSON 示例

```json
{
  "Camera1-1.avi": {
    "300": [
      [237, 503, 270, 568],
      [336, 573, 369, 624]
    ],
    "301": [
      [238, 504, 271, 569]
    ]
  },
  "Camera1-2.avi": {
    ...
  }
}
```
